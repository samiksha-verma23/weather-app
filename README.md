# 🌦️ Weather App

A responsive Weather App built using **HTML, CSS, and JavaScript** that displays real-time weather information using the **OpenWeather API**.

## 🚀 Features

- Search weather by city
- Real-time temperature, humidity, and wind speed
- Dynamic weather icons
- Responsive UI

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- OpenWeather API
